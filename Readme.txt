## Description du projet

Ce projet est un site vitrine basique réalisé en HTML, CSS et JavaScript. L’objectif était d’appliquer les bonnes pratiques de gestion de versions avec Git, en travaillant avec plusieurs branches et en effectuant des fusions progressives.

## Structure des branches

- main : version finale et stable du site
- dev : version intermédiaire d’intégration
- test/frontend : test des fonctionnalités avant validation
- feature/html-structure : structure HTML (pages de base)
- feature/css-theme : style CSS (design simple)
- feature/js-interactions : scripts pour les interactions

## Étapes principales

1. Initialisation avec git init
2. Création des branches de développement
3. Développement HTML, CSS et JS
4. Tests et fusions successives jusqu’à la branche main
5. Déploiement sur GitHub
